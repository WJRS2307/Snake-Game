# Snake-Game
Built using C# and Windows Forms using Object Oriented Programming. Guide the snake, eat the apples and see how long you can survive as the game speeds up across multiple levels.

## Controls
- **W**: Move Up
- **A**: Move Left
- **S**: Move Down
- **D**: Move Right

## Features
- **Progressive Difficulty**: 5 Distinct levels with snake's speed increasing as you advance
- **WASD Gameplay**: Navigate the game with WASD
- **Highscores**: Earn points by collecting as many apples as possible
- **Sound Effect**: Nostalgic background music

## How to setup:
1. Open SnakeTaskComplete.sln in Visual Studio.
2. Build and run the project (F5).
3. Enter your name when prompted
4. Click on START and PAUSE on the dashboard to control the gameflow.

## Highscore
1. Highscore file is located in D:\temp\Scores.txt
